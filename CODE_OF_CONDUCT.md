# Code of Conduct

This project is for people who want AI-assisted document generation to be more
truthful, verifiable, and respectful of private data.

## Expected behavior

- Be kind, direct, and constructive.
- Assume good intent, but be precise about risks.
- Keep discussions focused on truthful automation, privacy, documentation, and
  engineering quality.
- Use synthetic data when sharing examples.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Sharing someone else's private CV, contact details, job applications, or
  interview notes.
- Using the project discussion space to request or promote fabricated
  credentials, false claims, or deceptive job-application materials.
- Posting secrets, access tokens, or private employer/application data.

## Enforcement

The maintainer may edit, hide, or remove comments and issues that violate these
rules. Repeated or severe violations may lead to blocking from the project.
